# 🎙️ Kokoro TTS Studio

Персональная GUI озвучки на базе нейросети **Kokoro**. Позволяет генерировать высококачественную речь локально на вашей видеокарте. Идеально подходит для создания эдитов, озвучки игровых персонажей или контента для соцсетей.

---

## ✨ Основные возможности

* 🚀 **Полная автоматизация:** Скрипты сами установят Python, создадут окружение и скачают модели.
* 🎧 **Высокое качество:** Использование модели Kokoro (82M параметров) для естественного звучания.
* 🧩 **Длинные тексты:** Автоматическая склейка фрагментов — ИИ дочитает ваш сценарий до самого конца.
* 📂 **Архивация:** Все сгенерированные файлы автоматически сохраняются в папку `Saved voices`.
* 🎨 **Modern UI:** Удобный интерфейс на Gradio с выбором голосов и регулировкой скорости.

---

## 🛠️ Быстрый старт (Windows)

Для работы необходим установленный [eSpeak-ng](https://github.com/espeak-ng/espeak-ng/releases).

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/Ysak-Emir/Kokoro-TTS-Studio_WEB-GUI](https://github.com/Ysak-Emir/Kokoro-TTS-Studio_WEB-GUI)
    cd Kokoro-TTS-Studio_WEB-GUI
    ```
	**Или скачайте репозиторий (Download ZIP)**

2.  **Запустите установку с помощью скрипта:**
    Дважды кликните по `install_gui.bat`. 
    *Скрипт сам проверит наличие Python, создаст виртуальное окружение `studio_venv` и установит все библиотеки.*

3.  **Запустите скрипт:**
    Дважды кликните по `run_gui.bat`. Интерфейс откроется в браузере автоматически.

---

## 📂 Структура проекта

* `gui.py` — Основной код приложения и интерфейса.
* `install_gui.bat` — Мастер автоматической настройки "в один клик".
* `run_gui.bat` — Скрипт быстрого запуска.
* `requirements.txt` — Список зависимостей Python.
* `Saved voices/` — Папка с вашими готовыми озвучками.
